What is ShareCode?

ShareCode allows you to share nice looking code snippets with anyone. Just paste your code snippet and click Share. Then use the link to the snippet to share it with anyone you choose.

Syntax highlighting is provided for several different languages.

Code snippets can be a maximum of {sizeLimit}kb, will be stored for a maximum of one month and will only be available via the link you are redirect to after sharing.
